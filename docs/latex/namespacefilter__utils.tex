Kalman filter for sensor data. 

This is a recursive algorithm for estimating the internal state of a linear dynamic system from a series of noisy measurements. It is an optimal estimator in the sense that it minimizes the estimated error covariance. The Kalman filter is a two-\/stage algorithm\+:
\begin{DoxyEnumerate}
\item Prediction\+: The Kalman filter produces estimates of the current state variables, along with their uncertainties. These estimates are based on system dynamics and on the previous state estimate.
\item Update\+: The Kalman filter combines the current estimate with the new measurement information to produce an improved estimate of the current state variables along with their uncertainties.
