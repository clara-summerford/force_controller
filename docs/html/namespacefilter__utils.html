Applies a deadband to the input sensor data.

This is a control error band in which a system's response is insensitive to the magnitude of the error. The deadband is a range of values in which a sensor's output is zero. Use this to eliminate undesired control motion.

Exponential moving average filter for sensor data.

This is a type of infinite impulse response filter that applies weighting factors which decrease exponentially. The weighting for each older datum decreases exponentially, never reaching zero.

Kalman filter for sensor data.

This is a recursive algorithm for estimating the internal state of a linear dynamic system from a series of noisy measurements. It is an optimal estimator in the sense that it minimizes the estimated error covariance. The Kalman filter is a two-stage algorithm:

  1. Prediction: The Kalman filter produces estimates of the current state variables, along with their uncertainties. These estimates are based on system dynamics and on the previous state estimate.
  2. Update: The Kalman filter combines the current estimate with the new measurement information to produce an improved estimate of the current state variables along with their uncertainties.
    Parameters
    process_noiseThe process noise
    measurement_noiseThe measurement noise
    inputThe input sensor data
    Returns
    The filtered sensor data

    https://en.wikipedia.org/wiki/Kalman_filter
