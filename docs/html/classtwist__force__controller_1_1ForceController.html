Computes the twist based on the measured and target wrench.

It uses a PID controller to compute the twist value based on the force error. The PID controller is tuned using the parameters kp_, kd_ and ki_.

Note
This is implemented for both 1D and 6D force control via using vectors
Parameters
measured_wrenchThe measured wrench or force in the set dimension
Returns
The computed twist as a vector.

◆ isValidSensorWrench()

bool twist_force_controller::ForceController::isValidSensorWrench ( )
private

Checks to ensure the measured sensor wrench is valid.

It ensures the wrench doesn't get dangerously high or isn't zero. For high wrenches, it ensures the wrench magnitude does not exceed 90% of given max.

Note
. This is used for incoming force-torque data
Returns
True if wrenches are valid, false otherwise

◆ publishTwist()

void twist_force_controller::ForceController::publishTwist ( )
private

Publish twist_ values for force control.

It computes the desired twist based on PID control and the target force setpoint in every cycle iteration

Note
For safety, it checks the wrench magnitude does not exceed a 90% of given max, and clamps the twist value to the max twist value
Postcondition
The twist message is published
